The logo directory contains the Beedra logo in different formats.

The logo was created originally in Deneba Canvas 7 on a Macintosh (*.CNV). All other files are derived from that original.

The intention is to use higher quality JPEG files throughout the site, and not GIF files (barring evolutions in the patent world). However, I do not seem to be able to make a JPEG transparent, so in the cases were transparency is desirable, we use GIF instead.
